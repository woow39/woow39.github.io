This list will be styled like this, Goomba 1, 1. This first number means how many stomps it takes to get the reward. Anything after is the coin amount or reward. So it's something like this, stomps, reward.

  • Pink Yoshi = 1, the next 3 1 hit enemies give you 3 coins

Power Up Packs

Power up packs are cool. They change the way Lego Mario/Luigi plays. I'm actually only missing one of them. Here they are.

  • Fire Mario/Luigi
  • Cat Mario/Luigi
  • Builder Mario/Luigi
  • Propeler Mario/Luigi
  • Bee Mario/Luigi
  • Penguin Mario/Luigi
  • Tanooki Mario/Luigi

Now i will tell you what each of them do. The fire Mario suit allows you to flick Lego Mario to get coins, like he's shooting a fire ball.
The cat Mario suit allows Lego Mario to climb up walls for coins. The builder Mario suit allows Lego Mario to stomp for coins.
Next is the propeller Mario suit which allows you to jump with Lego Mario then shake him side to side for coins.
Now we have bee Mario. You jump then move him around in the air for coins.
Penguin Mario: Belly slide around for coins.
Last but not least we have Tanooki Mario. tilt him side to side for coins. this will also subtract 1 hit from enemies everytime you shake.

Adventures with Mario Starter Set Review

This set is one of the ones that you need to start your adventures with Lego Mario. It's one of the only set that comes with the Lego Mario figure, the course's start and the course's end.
The enemies included are, a Goomba and Bowser Jr.. It also comes with a ? block.
so overall this is a pretty cool set. It retails for about $70 in Canada

Whomp's Lava Trouble Set Review

This set is pretty small, but it does have more pieces than the desert pokey set, even though they both cost the same, about $25 in Canada.
This set has mostly lava (A.K.A. red) pieces for its ground pieces. There are only two enemies in this set. They are a Lava Bubble and a Whomp. It also has a P block. This set is pretty basic but cool.

Desert Pokey Set Review

Even though this set is smaller than the Whomp's Lava Trouble set, I like it more. The reason being the hammer platform thing you use to knock the Pokey segments down is really cool. You use it to knock over four Pokey segments, then step on one of them. It also comes with a Monty Mole. That is why this set is also really cool.

Desert Pokey Pictures

Guarded Fortress Set Review

This was my favorite Lego Mario set that I have. The reason I like it so much is because of the contraptions.
The first one is the Piranha Plant contraption. You jump on some sort of lever thing and it pushes the Piranha Plant into the pipe, and the barcode block pops up.
The second contraption is the flagpole thing. You jump on a plate and the flagpole pops up. the final contraption is the fortress. You jump on the Pow Block and the fortress door opens. You can also get coins from the door.
This set also comes with a Bob-omb and a Koopa Troopa. Overall this set is really cool and retails for about $70 in Canada.
